# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v0.4.x  | :white_check_mark: |
| < v0.4  | :x:                |

## Reporting a Vulnerability

**Do not open public issues for security vulnerabilities.**

If you discover a security issue, please email **security@verdict.dev** (or privately report via GitHub Security Advisories if enabled). We will triage and respond within 48 hours.

### Scope
*   **Verdict CLI**: Remote code execution (RCE) via malicious trace/config files.
*   **Verdict Action**: Command injection via inputs.

Testing against our public infrastructure is strictly prohibited.
